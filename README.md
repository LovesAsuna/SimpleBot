## 💽 如何使用

### 自编译

- 注意: 请使用 Cargo 打包

1.1. Clone 或者下载这个项目.

```bash
git clone https://github.com/LovesAsuna/SimpleBot.git
```

1.2. 编译

cargo run

### 直接下载

1. 到 [Release](https://github.com/LovesAsuna/SimpleBot/releases) 下载最新版本的 可执行文件
2. 根据对应的平台启动
